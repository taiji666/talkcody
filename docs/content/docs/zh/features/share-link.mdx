## 什么是分享链接？

分享链接功能允许你将 AI 对话任务生成一个专属链接，方便与他人分享你的工作成果。接收者无需安装 TalkCody 或登录账号，直接通过浏览器即可查看对话内容。

<Callout type="info">
分享链接适用于：代码审查、知识分享、教学演示、团队协作等场景。
</Callout>

## 主要特性

| 特性 | 说明 |
|------|------|
| **无需登录** | 接收者无需账号，直接在浏览器中查看 |
| **路径隐私保护** | 自动将本地文件路径（如 `/Users/xxx/`）转换为 `~`，保护隐私 |
| **工具输出展示** | 完美展示 readFile、writeFile、editFile、codeSearch 等工具的执行结果 |
| **密码保护** | 可选设置密码，防止未经授权的访问 |
| **过期时间** | 支持 1 天、7 天、30 天或永不过期 |

## 使用方法

### 创建分享链接

1. **打开分享对话框**
   
   在任意任务对话中，点击聊天面板右上角的 **分享图标** 按钮。

2. **配置分享选项**
   
   在弹出的分享对话框中，你可以设置：
   
   - **过期时间**：选择链接的有效期限
     - 1 天
     - 7 天
     - 30 天
     - 永不过期
   
   - **密码保护**（可选）：为链接设置访问密码
     - 勾选"启用密码保护"
     - 输入密码（至少 4 位字符）

   <Callout type="warning">
   请妥善保管密码，分享链接时需要将密码一并告知接收者。
   </Callout>

3. **生成并复制链接**
   
   点击 **创建分享链接** 按钮，系统将生成专属链接。
   
   你可以：
   - 点击 **复制链接** 将链接复制到剪贴板
   - 点击 **在浏览器中打开** 预览分享页面

### 查看分享页面

分享链接的接收者将看到：

- **对话标题**：任务的名称
- **模型信息**：使用的 AI 模型
- **消息列表**：完整的对话内容，包括用户提问、AI 回复和工具执行结果
- **代码块**：支持语法高亮和复制功能
- **工具输出**：以友好的格式展示文件操作、搜索结果等

<Callout type="tip">
代码块右上角有复制按钮，点击即可复制代码内容。
</Callout>

## 分享内容说明

### 会被分享的内容

- 对话中的所有消息（用户和 AI）
- AI 使用的工具调用及其输出结果
- 代码块、文件路径、搜索结果等

### 不会被分享的内容

- 本地文件的完整路径（自动转换为 `~`）
- 敏感的命令输出（如 API 密钥、Token 等）
- 你的个人账号信息

## 最佳实践

1. **敏感信息**：分享前检查对话内容，确保没有泄露敏感信息
2. **密码保护**：对于敏感内容，建议启用密码保护
3. **合理过期**：根据用途选择合适的过期时间
4. **及时清理**：不再需要的分享链接请及时删除
